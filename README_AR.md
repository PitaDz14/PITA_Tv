# تعليمات سريعة - PITA DZ TV

## البدء السريع

### الخطوة 1: تثبيت Node.js
إذا لم يكن مثبتاً بعد، حمّل Node.js من:
**https://nodejs.org/**

اختر النسخة LTS (الموصى بها)

### الخطوة 2: تثبيت الحزم
افتح **Command Prompt (CMD)** (ليس PowerShell) في مجلد البرنامج واكتب:
```
npm install
```
انتظر حتى يكتمل التثبيت (قد يستغرق بضع دقائق)

### الخطوة 3: تشغيل البرنامج
```
npm start
```

## بناء ملف EXE للتوزيع

بعد تثبيت الحزم، اكتب:
```
npm run build
```

ستجد الملف التنفيذي في مجلد `dist`

## ملاحظات مهمة

- ⚠️ استخدم **CMD** وليس PowerShell للتثبيت
- ⚠️ تأكد من اتصالك بالإنترنت أثناء التثبيت
- ⚠️ قد يستغرق التثبيت 5-10 دقائق
- ✅ بعد التثبيت الأول، التشغيل سيكون فورياً

## حل مشكلة PowerShell

إذا واجهت خطأ في PowerShell، استخدم CMD بدلاً منه:
1. اضغط Win + R
2. اكتب: cmd
3. اضغط Enter
4. انتقل لمجلد البرنامج: cd "C:\Users\Administrator\Desktop\PITA TV"
5. نفذ الأوامر
